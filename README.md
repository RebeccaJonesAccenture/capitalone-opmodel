# Learning & KM Operating Model

Interactive, single-page visualization of the Learning & Knowledge Management
operating model.

## Run

```bash
npm install
npm run dev
```

Vite serves on http://localhost:5173.

## Stack

Vite + React 18 + TypeScript, Tailwind CSS for styling, Framer Motion for the
expand/collapse animations. All content lives in
`src/components/OperatingModel.tsx`.
